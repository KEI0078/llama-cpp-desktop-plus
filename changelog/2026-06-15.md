## 🔧 saveConfig 整合机制

```
用户在前端"独立设置页"调整参数
  → system.json 存 { enabled: { "cat-X.name": true }, values: { "cat-X.name": XX } }
  → 保存配置 / 启动服务
  → 读取 system.json + params/*.json flag 映射
  → 遍历 enabled params → 用 flagMap.get(shortName) 得到 --flag
  → 拼接成 "--threads 8 --ctx-size 32768 --mmap ..."
  → 追加到 config.extra_args
  → 写入 config.toml
  → llama-server 启动时带上这些参数
```
